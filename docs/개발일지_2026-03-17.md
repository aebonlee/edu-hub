## 4. 로그인 필수 링크 (Login Gate)

### 4-1. Courses.jsx 외부 링크 게이팅
- **수정**: `src/pages/Courses.jsx`
- 기존 `<a href={ls.url}>` → `<button onClick={handleVisitSite}>`
- `handleVisitSite(url)`:
  - 미로그인 → sessionStorage에 returnUrl 저장 → `/login`으로 리다이렉트
  - 로그인 상태 → `window.open(url, '_blank')` 실행

### 4-2. Login.jsx returnUrl 처리
- **수정**: `src/pages/Login.jsx`
- `location.state.returnUrl` 수신
- 이메일 로그인 성공 시 returnUrl이 있으면 `window.open(returnUrl)` 후 페이지 이동
- OAuth 콜백 후: `isLoggedIn` 감지 시 sessionStorage의 `dreamit_return_url` 확인 → 외부 사이트 열기
- sessionStorage 사용으로 OAuth 리다이렉트 시에도 returnUrl 유지
